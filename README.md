# salaf-hub

A lightweight browser football game inspired by the feel of FC Mobile / eFootball.

## Run locally

Because this is a static web app, you can run it with any simple file server:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173` in your browser.

## Features

- Team naming and difficulty selection
- Real-time match simulation with AI opponents
- Player control (move, sprint, shoot/tackle)
- Scoreboard, match timer, pause/resume, and full-time result

## Controls

- Move: `W A S D` or Arrow keys
- Sprint: `Shift`
- Shoot/Tackle: `Space`
- Pause: `P`
